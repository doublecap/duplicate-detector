README

Vaatimukset: python 3.12 tai suurempi

Kuvaus: Skripti vertailee juurikansioon sisältyviä kuvia (voivat olla omissa kansioissaan) niiden pikseliarvojen keskiarvojen kautta ja kerää csv-tiedostolle duplikaattien nimet ja polut.

Asennusohjeet:
- Avaa komentorivillä hakemisto, johon haluat skriptin asentaa
- kun komennot on ajettu, kopioi main.py kansioon
- Aja seuraavat komennot:

pipenv install
pipenv shell
pip install pillow

- avaa main.py ja aseta root_folder ja output_csv -muuttujiin polut
- huom. 
- palaa komentoriville ja aja seuraava komento

py main.py
